Инструкция по установке:
1.Если без деплоя на сервер, запуск с обычного компьютера:
    1.1 Скачать Пайтон, версия 3.12 из интернета. При установке обязательно проставить галочку "Add to PATH" в начале установки должно быть.
    1.2. Скачать из интернета PyCharm. Открыть проект с помощью пайчарма. Активировать Виртуал Энв(скину с venv проект), поэтому в открытом пайчарме, слева сбоку снизу найти иконку выглядящую как квадратик, в середине которого >_ (это терминал.) в терминале, перед строкой где стоит курсор должен быть написан (.venv) , по типу "(.venv) S. C:\Users\Projects\Temka_tg>" если все так - супер. Если нет – напиши в терминале .\.venv\Scripts\activate.
    1.3 Пропиши в терминале следующую команду pip install -r requirements.txt
    1.4 Открой файл ".env" в поле TELEGRAM_API_KEY после =без пробелов впиши свой токен(сейчас я поставил тот токен, который ты мне дал)
    То же самое проделай с ADMIN_ID. Твой айди был (789958154) но перепроверь.
    То же самое проделай с адресом бд.

        заполни в .env файле следующие поля(DB_NAME=0
        DB_HOST=0
        DB_PASSWORD=0
        DB_USER=0) вместо 0 твоих данных. Без пробелов!!!

    1.5 Пропиши команду python .\db\models.py
    1.6 Пропиши команду python main.py
